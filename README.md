# TriView
> 여행 플래너 프로젝트
개인의 경험들을 참고해 일정을 기획할 수 있는 여행 플래너의 필요성을 느껴 프로젝트를 진행했습니다.

* 여행 플랜을 쉽게 만들고 저장할 수 있다.
* 회원 기능을 통해 내 여행플랜을 생성, 조회, 수정, 삭제할 수 있다.
* 메인 페이지에서 오늘의 날씨와 지역별 Google 추천 명소로 이동 할 수 있다.

# 사용방법
> 현재 구글 북마크를 바로 플래너에 가져올 수 있도록 수정중입니다.
